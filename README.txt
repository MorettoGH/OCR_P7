Pour faire fonctionner l'application Groupomania veuillez suivre les étapes suivantes :

1) Démarrer le serveur Backend.
- Se placer sur le repertoire nommé backend
- Ouvrir un terminal et taper la commande suivante : node server

2) Démarrer le serveur Frontend
- Se placer sur le repertoire nommé frontend
- Ouvrir un terminal et taper la commande suivante : npm start

Les technologies utilisées sont Node.js et express.

Pour la mise en place de la base de donnée MongoDB, vous référer au README fourni avec le dossier correspondant (moretto_christophe_1_bdd_082022)
